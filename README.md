# SkillFlow AI 赋能服务平台

这是一个静态官网项目，可直接部署到 GitHub Pages。

## GitHub Pages 发布方式

1. 打开 https://github.com/new
2. 创建公开仓库，例如 `skillflow-ai-platform`
3. 上传本目录里的文件：
   - `index.html`
   - `styles.css`
   - `script.js`
   - `assets/hero-ai-skill-platform.png`
   - `.nojekyll`
4. 打开仓库的 `Settings > Pages`
5. `Source` 选择 `Deploy from a branch`
6. `Branch` 选择 `main`，目录选择 `/root`
7. 保存后等待 1-3 分钟

最终网址格式：

```text
https://你的GitHub用户名.github.io/skillflow-ai-platform/
```
